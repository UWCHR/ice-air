    Description:
      Abstract:
        This map layer portrays the county boundaries of the United States, Puerto Rico, and 
        the U.S. Virgin Islands as linework.  The map layer was derived from the 1:1,000,000-Scale County 
        Boundaries of the United States polygon data set by applying the Arc/Info Clean function.  The 
        Clean function eliminates the duplicate arcs present in polygon shapefiles, leaving one copy of 
        each line.
      Purpose:
        These data are intended for geographic display and analysis at the national level, and for large 
        regional areas.  The data should be displayed and analyzed at scales appropriate 
        for 1:1,000,000-scale data.  No responsibility is assumed by the National Atlas of the United 
        States in the use of these data.
      Supplemental_Information:
        This data set was derived from the July 2012 version of the County Boundaries of the United States 
        polygon data set; that data set has since been revised.  As a result, there is a discrepancy 
        between the coastline depiction in this data set and in the current County Boundaries polygon data 
        set.  The discrepancy is limited to a part of the St. Croix River that separates Maine from the 
        province of New Brunswick in Canada. 

        The County Boundaries polygon data set locates the coastline about 14 km downstream from where it 
        is shown in this County Boundary lines data set.  The polygon version matches the networked 
        1:1,000,000-scale Streams data set. 

        This data set, with a coastline that extends 14 km further upstream, matches the National 
        Atlas 100-meter raster data sets.

        All County, State, and National boundaries agree between the polygon and line versions of this data 
        set. There are no current plans to resolve the coastline discrepancy.

      Process_Step:
        Process_Description: 
          The National Atlas 1:1M Counties 2014 polygons were converted to an Arc/Info coverage, then the 
          Clean command was run with a dangle length of 0, a fuzzy tolerance of 1.0 meters, and an output 
          feature type of LINE.  The resulting output coverage was converted to a shapefile and fields for 
          Linecode (integer) and Linetype (text) were added to the attribute table.  The shapefile was 
          edited to set values that distinguish coastlines and boundary types, then cartographically 
          reviewed to fix errors and omissions.  The Canadian shoreline of the Great Lakes was from the 
          Canada shoreline data set.
